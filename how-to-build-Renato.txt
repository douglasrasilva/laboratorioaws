Em cada um dos servidores
Acesse o Host via SSH e use os comandos abaixo:

sudo su -

echo "172.31.X.X elastic0" >> /etc/hosts
echo "172.31.X.X elastic1" >> /etc/hosts
echo "172.31.X.X elastic2" >> /etc/hosts
echo "172.31.X.X elastic2"  >> /etc/hosts


Vamos preparar as maquinas, e lembre-se de substituir o "X" pelo valor correspondente!

  
apt install net-tools

hostnamectl set-hostname elasticX
  
cd / && mkdir webtools
  
chown -R ubuntu:ubuntu webtools
  
###Inserir entradas no arquivo (Pré requisito do produto)

vim /etc/security/limits.conf 


#Configurações para o ELK - Colar conteúdo abaixo
* soft nofile 65536
* hard nofile 65536
  

###Inserir entradas no arquivo (Pré requisito do produto)
vim /etc/sysctl.conf 

#Configurações para o ELK - Colar conteúdo abaixo
vm.max_map_count=262144

reboot
